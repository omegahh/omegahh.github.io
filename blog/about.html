About Me

一名兴趣广泛的 90 后，数学系毕业，目前研究生阶段的研究方向是机器学习 (Machine Learning) 和数据挖掘 (Data Mining)，主要是应用在生物数据上，就是生物信息学啦 :)。由于我的数学背景，所以我的研究重点是放在机器学习理论的研究上，尤其是优化方法和模型原理的研究，嗯，研究生阶段我还是计划打好基础，多深入到理论的研究中去。同时我是一个开源爱好者，平时喜欢折腾，喜欢 Linux, Python，算是一个熟练的 Linuxer 吧。

About Blog

这个博客创建于 2015 年初，是一个基于 Jekyll 的静态博客，模版主题名为 iLotus。博客的主题其实我想重新起名为 Alice 的，可是博客中我最喜欢的三个设计元素：切换主题、宽屏双栏和适合阅读的风格，都是来源于 PiZn。虽然我在其提供的模版上，一行一行的把代码都过了一遍，确实细致到 JS 的每一个函数，CSS 里面的每一行，全部改成了我的代码风格。又添加了些许功能，但是还是需要感谢原创者 PiZn，第一次看到这个风格简洁，有着阅读书卷感觉的排版时，就决定用这个 CSS 了，简直太赞！

博客的风格几乎没有改变，但是在功能上有了很多改进和添加，当然本博客主题的代码遵循 MIT 协议，有心的人可以在我的 Github 上找到源代码。目前本博客的服务器地址在亚马逊的免费服务器 AWS 上，考虑搬迁中…

更改纪录：

  • 1. 代码高亮：使用的是 prism.js 支持，将原来的 code 样式进行了修改以匹配代码样式；

  • 2. 图片修正：原始模版对图片排版有些许瑕疵，目前改良了显示，并且将小图按原规格显示并居中；

  • 3. 数学公式：使用 MathJax 提供了对数学公式的支持；

  • 4. 切换色调：修正了切换拍版色调的功能，导航栏心形按钮启用色调切换；

  • 5. 搜索功能：添加一个快速搜索功能，导航栏搜索按钮或者双击 Ctrl 键启用，且支持 ESC 退出；

  • 6. 其它修改：主要是 CSS 和 JS 代码的重写，去除了大量的冗余代码，以及自己不喜欢的排版风格；
