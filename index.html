Bumble Bee

Bumble Bee is a static blog built by Jekyll (a powerful framework could compile static blogs from markdown files automatically.). At the begining, I spend too much time in webpage design while no blog was posted. I realized that my main job should be writing more blogs. Thus I chose a suitable template and focus on writing quickly. iLotus is my final choice. It's a beautiful theme forked from Pizn. And it is simple and comfortable for reading. Additionally, I also add many new custom features, like code highlight, quickly search and math formulation supports, to enrich its functions. Within these powerful tools, I really enjoy the processes of writing blogs.
